1.git init - inicijalizacija na git vo direktorium
2. git add . - dodavanje na promenite sto se napraveni vo version control
3. git push - isprakjanje na promenite na github
4. git checkout -b "bug" - kreiraj granka "bug"
5. git branch - proverka na koj branch se naogjate
6. git checkout master - prefrli se na taa granka
7. git checkout -b "bug2" - kreiraj granka bug2
